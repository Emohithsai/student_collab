# Student Group Project Management Platform

## Project Overview
A React-based frontend platform designed to help students and teachers coordinate group projects.

## Tech Stack
* **Frontend:** React.js
* **Routing:** React Router DOM
* **State Management:** React Hooks (useState)

## Features (Review 1)
* **Authentication UI:** Role-based login (Student/Teacher).
* **Student Dashboard:** Task tracking and milestone monitoring.
* **Teacher Dashboard:** Group progress overview and status tracking.
* **Navigation:** Seamless page transitions.
